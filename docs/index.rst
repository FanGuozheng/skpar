======================================================================
SKOPT
======================================================================

`SKOPT` is a tool that helps to optimise Slater-Koster (SK) files for 
use by Density-Functional-based Tight Binding (DFTB) programs like 
`DFTB+`_ (BCCMS Bremen), Lodestar_ (The University of Hong Kong), 
QuantumWise_ and possibly other implementations of DFTB. 
(See `dftb.org`_ for details on available SK data sets)

`SKOPT` is written in `Python`_ and is still in development. 
It is available under `The MIT License`_.

The `Documentation`_ contains more details.
